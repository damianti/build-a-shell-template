# myshell

> Write your README below. This file is graded (4 pts) — an empty or incomplete README loses points.

## What to include

Your README must cover **all** of the following:

- **Project purpose** — one or two sentences explaining what this shell does and what you built
- **How to build** — the exact commands to compile from a clean directory:
  ```bash
  cmake -S . -B build
  cmake --build build
  ```
- **How to run** — the command to launch your shell after building (`./build/myshell`)
- **File descriptions** — a short description of every source file you wrote and its role in the project
